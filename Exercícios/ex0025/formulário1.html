Testanto Formulários

Algumas Anotações:

Todo o formulário deve estar dentro da tag form.

A propriedade "action" refere-se para onde vai os dados inseridos no formulário, já a propriedade Method, de maneira resumida, refere-se a URL se os dados inseridos vão ou não estar contidos na mesma.
A tag label serve para interligar o conteúdo com a descrição dele.
O input possibilita a interação com o usuário, possibilitando a inserção dos dados do usuário. Suas propriedades limitam o que esse input vai aceitar.
